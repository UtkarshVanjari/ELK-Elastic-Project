# Elastic Stack Project (Phase 1 & Phase 2)

This project demonstrates hands-on setup and secure integration of
Elasticsearch and Kibana on a Linux environment.

## Phase 1 – Setup & Security
- Installed Elasticsearch & Kibana
- Enabled security features (TLS & authentication)
- Tuned JVM and system settings
- Verified cluster and service health

## Phase 2 – Integration & Validation
- Connected Kibana with Elasticsearch securely
- Tested APIs via CLI
- Analyzed logs and resolved configuration issues
- Ensured service stability
